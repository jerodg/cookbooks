= Windows Cookbook

== Create Symbolic Link
=== Folder
To create c:\test\my_folder as link to c:\some_folder\my_folder
[source,bash]
mklink /J <link-path> <source-path>
mklink /J c:\test\my_folder c:\some_folder\my_folder
